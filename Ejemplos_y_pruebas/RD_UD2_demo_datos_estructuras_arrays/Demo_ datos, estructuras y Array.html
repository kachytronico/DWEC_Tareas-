Demo: datos, estructuras y Array


En esta demo te muestro en funcionamiento ejemplos de todos los conceptos que acabas de ver: tipos de datos, estructuras de control y Arrays.

Pulsa el botón derecho de tu ratón en esta web y "Ver código fuente de esta página", ahora fíjate cómo la etiqueta "script" hace referencia la archivo "JavaScript.js", ya sabes del tema anterior que se trata de un archivo js externo.

Pulsa encima de "JavaScript.js" y verás una secuencia de funciones, una detras de otra.

Vuelve al código fuente de cada una de las páginas (Conceptos básicos, Variables, Arrays y Funciones)y busca con CTRL+F la palabra "onclick".

Verás que aparece varias veces en cada página y que se iguala a alguna de las funciones del archivo js. Por ejemplo: onclick="usoComillas()"

De esta manera comprobarás que al hacer clic en los distintos cuadros grises se ejecuta realmente el código que tienen dentro y muestra el resultado en un alet.

Navega por las distintas páginas y depura el archivo "JavaScript.js" para comprobar los ejemplos.
